Tools for mentors and organizers of WordCamps.

== Description ==

Current tools:

= Planning Checklist =
* Provides a list of tasks that need to be completed during the planning and execution of a WordCamp.
* Tasks can be marked as complete or skipped to keep track of progress.
* Tasks are organized into different categories that correspond with common organizer roles.
* Mentors and organizers can collaborate on the checklist, and changes will be synced between each user viewing it.
